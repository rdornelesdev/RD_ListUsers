# 🚀 RD_ListUsers
C# .NET console app to list and view user data using a layered architecture.

🎯 Project Goal:
The primary objective of this project was to transition from procedural code to Clean Architecture, enforcing separation of concerns and leveraging advanced C# features.

🏗️ Architecture:
The project is structured according to Clean Architecture principles:
 - Domain: Contains core entities (User) and pure business rules.
 - Application: Manages service logic and data retrieval.
 - ConsoleApp: The presentation layer, handling terminal-based user interaction.
